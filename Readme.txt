# Chatbot-with-Blockchain-using-RFID
## Description
We are making a chatbot based on block chain using ARK api and amazon lex with contact tracing. The API is stored in MongoDB. The function of this chatbot is to match the symtops provided to the chatbot by user in its database API from a current medical databse. If the symtoms match the patient is advised to go to the doctor or else it advises for preliminery precautions.
